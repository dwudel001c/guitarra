== Success and Error codes

.Success and Error conditions 
|===
| Http Code | Description | Notes

|200 
| success
| All OK


|400 
| invalid input
| Covers Various conditions where client did not provide correct value or parameters


|403 
| Good Luck with that
| 

|404 
| no found
|

|405 
| Method not allowed 
|

|500 
| socket, timeout errors, DB errors. 

|  
|===
